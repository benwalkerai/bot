# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of Inzen CLI Bot.

## Reporting a Vulnerability

If you believe you have found a security issue, please do not open a public issue.

Report it privately through the GitHub Security Advisories workflow for this repository, or contact the maintainers through the issue tracker with a request to switch the discussion to a private channel.

Please include:

- A short description of the issue
- The affected version
- Steps to reproduce
- Any relevant logs or screenshots
- Whether the issue affects local data, network traffic, or exported output

## Response Expectations

We aim to acknowledge reports promptly, confirm scope, and provide a fix or mitigation as soon as practical based on severity.

## Safe Disclosure

Please avoid sharing API keys, tokens, or sensitive session data in the report itself. Redact secrets before attaching examples whenever possible.
